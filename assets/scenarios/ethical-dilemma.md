# 场景预设：伦理 / 价值取舍

## 触发关键词
"该不该举报" / "面对道德两难" / "在A和B中如何选择是对的" / "我的做法对吗"

## 推荐角色配置（5 角色 standard mode）
1. Steelman 派（构造对立伦理立场的最强版本）
2. 第一性原理派（剥离社会期待找底层价值锚点）
3. Bayesian 更新派（不同决策的长期后果概率分布）
4. 怀疑派（隐性的道德陷阱）
5. 用户派（关系到的所有人的立场还原）

## 推荐 mode
`standard`（3 轮 + 红队）

## 议题重述格式
`【议题】在 [伦理张力] 中应该选择 [立场A] 还是 [立场B]？涉及 [价值冲突]`
`【类型】判断型，含价值层冲突`
`【成本】中-高（不可逆 + 影响声誉与自我认同）`

## required_inputs（v0.3.0 强制门禁）

```yaml
required_inputs:
  - 涉事各方：直接当事人 + 间接受影响者（未被提及的人也要点到）
  - 已造成 / 未造成的事实损害：实际发生 vs 假设性
  - 法律边界：是否触及法律 / 行业规范 / 公司制度
  - 用户最看重的 2-3 个价值锚点：公平 / 忠诚 / 自由 / 关怀 / 真诚 / 尊严 / 等
  - 已尝试的妥协方案：避免重复造轮子
```

## 输入预处理（兼容旧版口径）
让用户给出：
- 具体情境 + 涉及人物
- 用户内心的两个/多个声音分别在说什么
- 已尝试的妥协方案
- 用户最看重的 2-3 个价值（公平 / 忠诚 / 自由 / 关怀 / 真诚 / 尊严⋯）

## 输出额外要求
- **多伦理框架对照**：从功利主义 / 义务论 / 美德伦理 / 关怀伦理 4 个角度各做判断
- **价值优先级排序**：基于用户自陈价值，给出最一致的选择
- **后果链推演**：每个选项的 1 年后/5 年后情境

## ⚠️ 强制提示（顶部）
> 伦理决策**没有客观正确答案**。本辩论提供多视角和后果分析，最终判断和承担责任的是你自己。

## 已知陷阱
- **道德优越感**：用户希望听到"你是对的"的回应——必须警惕
- **自利性合理化**：人倾向给自己已倾向的选择找伦理理由
- **简化道德**：复杂情境不能简化为黑白
- **忽略受影响他人**：决策的后果常常波及未被提及的人
